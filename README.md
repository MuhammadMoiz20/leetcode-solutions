# LeetCode Solutions

My solutions to LeetCode problems in Python.

## Topics Covered

### Arrays and Strings
- Two Sum, Valid Parentheses, Reverse Integer
- Longest Substring Without Repeating Characters
- Merge Intervals, Trapping Rain Water

### Linked Lists
- Add Two Numbers, Linked List Cycle
- Merge Two Sorted Lists, Intersection of Two Linked Lists

### Trees
- Inorder/Level Order Traversal
- Validate BST, Lowest Common Ancestor
- Serialize/Deserialize Tree

### Dynamic Programming
- Climbing Stairs, House Robber
- Coin Change, Longest Increasing Subsequence
- Maximum Subarray (Kadane's Algorithm)

### Backtracking
- Generate Parentheses, Permutations
- Combination Sum, Subsets

### Graphs
- Number of Islands, Clone Graph
- Course Schedule, Word Search

## Usage
Run individual solutions from each file.
